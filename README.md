# 日めくりタスク

シニア向けの、カレンダー形式で見られるシンプルなタスク管理アプリの静的プロトタイプです。予定表にタスクを押し込むのではなく、日付ごとの「やることの箱」として見せることを優先しています。

## 使い方

`index.html` をブラウザで開くだけで動きます。

```bash
open senior_simple_tasks/index.html
```

ローカルサーバーで確認する場合:

```bash
cd senior_simple_tasks
python3 -m http.server 4173
```

その後、`http://localhost:4173` を開きます。

## 主な機能

- 日付を選んでタスクを追加
- 1週間分のタスクを日付ごとのカレンダー形式で表示
- 未完了数、今週の作業量、完了数のサマリー
- 完了、未完了への戻し、翌日への移動、削除
- 朝、昼、夕方、夜の時間帯指定
- localStorage への自動保存
- 文字を大きくする表示切り替え

## 設計メモ

- シニア利用を想定し、最小タップ領域はおおむね54px以上にしています。
- 時間割ではなく、日付ごとのタスク量と状態が見える構成にしています。
- タスクがない日にも「この日に追加」を置き、日付を選ぶ操作を減らしています。
- 説明文を増やしすぎず、画面上の次の操作が見える構成にしています。
